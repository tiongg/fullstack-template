# Fullstack template

Using nx repo and npm

## Packages

1. Backend (Nest.js)
2. Frontend (Next.js)
3. Utils

## Other stuff

1. docker-compose (postgres db)
2. Dockerfile (BE)
3. Dockerfile (FE)

## Directory structure

> [!TIP]
> Apps directory is used for stuff that will end up being the final product
> Packages directory is for stuff that apps depends on

```
apps/
  - backend/
  - frontend/

packages/
  - ui/
  - utils/
```
